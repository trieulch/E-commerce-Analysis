# E-Commerce Analytics | SQL End-to-End Business Analysis

## Project Overview
This project analyzes a real-world structured **e-commerce dataset** using **MySQL** to answer critical business and product questions across marketing, website performance, and product strategy.

The objective is to simulate the role of a **Data Analyst supporting Marketing, Product, and Executive stakeholders**, delivering actionable insights to drive revenue and conversion improvements.

---

## Executive Summary

Key insights identified:

• Website traffic grew consistently, driven primarily by **paid search (Gsearch)**  
• Conversion rate improved over time due to **landing page and billing page experiments**  
• Clear **seasonality patterns**, with strong growth toward Q4  
• Product expansion significantly increased **revenue per session**  
• Repeat users demonstrated **higher conversion efficiency**

## Dataset Structure

Main tables:

- `website_sessions` → traffic and acquisition data  
- `orders` → transaction data  
- `order_items` → product-level sales  
- `website_pageviews` → user behavior tracking
