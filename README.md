# E-Commerce Analytics | End-to-End SQL Business Analysis

## Overview

This project performs a comprehensive **end-to-end analysis of an e-commerce business** using **MySQL**, replicating real-world analytical workflows used by Data Analysts in technology companies.

The dataset (provided by Maven Analytics) contains realistic transactional, behavioral, and marketing data, enabling deep analysis across:

- Traffic acquisition
- Website performance
- Conversion funnel optimization
- Marketing channel performance
- Product performance
- User retention
- Business growth and seasonality

The goal is to answer stakeholder questions and generate insights to support **marketing, product, and executive decision-making**.

---

# Dataset Structure

Core tables:

- `website_sessions` — traffic source, device, and session info
- `website_pageviews` — user navigation behavior
- `orders` — transaction records
- `order_items` — product-level details
