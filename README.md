# E-Commerce Analysis in MySQL

## Overview
This project analyzes an e-commerce dataset using **MySQL** to uncover insights related to traffic acquisition, website performance, channel effectiveness, product performance, and user behavior.

The dataset is provided by **Maven Analytics** and reflects a realistic e-commerce database structure, including website sessions, orders, products, and traffic sources. The analysis replicates real-world business questions typically asked by marketing, product, and executive teams.
